Danny英文摘要

可折叠面板 · 完整英文提案摘要

BD Agreement (Community Development)

  • Base salary: $0/month (Phase 0 trial)
  • Commission: $150 per Effective Host
  • No monthly cap on commissions
  • Settlement: Monthly, paid by the 5th of next month
  • No clawback clause

Effective Host Definition (21-Day Verification)

  • Tier 1 (Day 1-7): Open a game with 5+ real players
  • Tier 2 (Day 1-14): 5+ games in 2 weeks, each game at least 1 hour
  • Tier 3 (Day 1-21): Host brings 10+ active users (50+ hands/week each)
  • Note: Effective Host ≠ Club. One club can have multiple effective hosts.

Active Player Volume Bonuses (per host, one-time)

  • Host brings 20+ simultaneous active players: +$100
  • Host brings 40+ simultaneous active players: +$200
  • Host brings 80+ simultaneous active players: +$400
  • Quarterly top BD award: +$200 (when multiple BDs)

Content Agreement (Optional, Separate Contract)

  • Base: $150/month
  • Deliverables: 4 short videos + 2 social media posts per month
  • Quality bonus: +$100 if all pass review
  • Monthly cap: $250

Phase Progression

  • Phase 0 (0-3mo): $0 base + $150/host. Unlock Phase 1 with 3 hosts + 50% survival
  • Phase 1 (3-6mo): $200/mo base + $180/host + regional exclusive rights
  • Phase 2 (6mo+): $300/mo base + $200/host + event co-sponsorship
