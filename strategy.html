产品定位
语音社交 x 德扑场景 x 房主生态
FIVE 是一款面向巴西市场的语音社交产品，以德州扑克为核心场景切入，构建以"语音房间"为基础单元的社交生态。
不是一个扑克工具，而是一个以扑克为"社交货币"的语音社区。

竞争格局 · 三类战场
战场一 · 语音社交
Yalla / StarMaker
Yalla巴西渗透低，FIVE本地化更深。StarMaker以K歌为主，FIVE以游戏社交切入。语音社交赛道尚无统治性玩家，窗口期存在。
战场二 · 线上扑克
PokerStars / PPPoker / Suprema
PokerStars偏竞技，PPPoker俱乐部制渗透高，Suprema本地化最深。FIVE差异化在于语音原生集成，社交体验远超纯扑克工具。
战场三 · 泛社交娱乐
WhatsApp群 / Discord / 线下棋牌
WhatsApp是天然传播通道也是迁移壁垒。FIVE提供结构化游戏场景，突破地理限制，比Discord更轻量更娱乐化。
核心差异化
语音社交 x 扑克场景 x 房主生态 = FIVE独占区域
纯扑克App不具备语音社交深度 · 纯语音App缺乏结构化娱乐场景
FIVE在交叉点建立壁垒：房主同时是社交组织者和游戏组织者
